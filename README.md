# Disease-diagnosis-
THIS IS A SIMPLE AI DIGNOSIS PROGRAM. INSTEAD OF JUST GIVING ONE ANSWER,IT LETS YOU ADD SYMPTOMS UNTIL FULLY DIAGNOSED.

# Overview
the disease diagonosis bot takes a set of symptons as input and uses a trained machine learning model to predict the most likely disease . it helps user get an intial understanding of potential health conditions.

machine learning classification model
data se driven analysis
implemented in both python and jupiter notebook
includes screenshots and demo video

getting started 
1. clone the repository
 git clone <your-repo-link>
cd disease-diagnosis-bot

2. install dependencies
 pip install pandas numpy scikit-learn

3. run the project
   python code.py

how it works 
1.user inputs sumptoms
2.symptoms are processed and encoded
3.model compares input with trained dataset
4.predicts the most probable disease
5.display the result
